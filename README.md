# EU Real Estate Market Analysis 🇪🇺

##  Project Overview
This project analyzes the real estate market in the European Union. The goal was to clean messy data, handle missing values, and provide insights into pricing trends and property types across different regions.

## 🛠️ Tech Stack
* **Python (Pandas/NumPy):** For data cleaning and handling missing values using nested grouping and median imputation.
* **Power BI (DAX/Power Query):** For building an interactive dashboard.
* **Excel:** Initial data source.

## 🧹 Data Cleaning Process
* Handled missing values in "Square Meters" and "Floor Number" using **Median Imputation** based on property type and location.
* Removed duplicates and standardized categorical columns.
* Performed **Outlier Detection** to ensure data quality.

## 📊 Key Insights
* Identified the top 3 most expensive cities for real estate in the EU.
* Discovered a strong correlation between property type and price per square meter.
* Visualized the distribution of property types (Apartments vs. Houses).
